# sync-local-to-nas

Script de synchronisation rsync vers NAS.

## Installation

```bash
git clone https://github.com/germain-italic/sync-local-to-nas.git
cd sync-local-to-nas
cp .env.example .env
```

## Configuration

Éditez `.env` avec vos chemins et hôte NAS.

## Utilisation

```bash
./sync.sh
```
